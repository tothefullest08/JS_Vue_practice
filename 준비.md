# 실습 UI 개발로 배워보는 순수 javascript 와 VueJS 개발(쇼핑몰 검색 페이지 만들기)



# 준비

## 개발 환경 구성

- Node.js
- 에디터: VS CODE
- Git (코드형상관리)
- Chrome 최신버젼
- Lite Server - 개발 서버. 홈페이지를 개발시, 개발용 서버로 사용되는 툴
  - Global 설치

```bash
$ npm install -g lite-server
 
# To run: 
$ lite-server
```



## 요구사항 분석 

- 검색폼 구현
- 검색 결과 구현
- 탭 구현
- 추천 검색어 구현
- 최근 검색어 구현
